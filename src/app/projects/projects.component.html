Coverage Control


The key idea behind the coverage algorithm is to assign a region for each agent to be responsible for. The idea of the implementation is to enhance the algorithm to make it more adaptive to a situation where there could be an obstacle in the map where the agents are being deployed.

Smart Museum

The paper focussed on designing and validating a context aware content delivery system using a Raspberry Pi which plays the role of a beacon. A local database had been set up to hold the required contents, this would be queried according to the ID of the beacon. The system provided a non-traditional way of Content delivery System that delivers the cultural contents.

Brain Net

The project investigates more on how the data from the EEG sensors (provided before-hand) can be processed in both the local fog server and the remote cloud server. The results were represented by comparing the time taken by the selected server to complete the processing.
